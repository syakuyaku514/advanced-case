概要説明
飲食店予約サービス
会員登録者様は飲食店の予約（日付・時間・人数）ができ、会員登録をしていない場合も飲食店の店名・地域・ジャンル・概要等を見ることができます。
会員登録者様はお気に入り登録をすることで、自分の好みに合った飲食店を探しやすくすることもできます。


目的
自社で飲食店予約サービスを運営
